系統異常

  • 當無法輸入中文時，且WINDOWS出現IME已停用，該如何解決？

    • 由 WINDOWS 左下角 [開始] → [設定] (齒輪圖示) 進入設定畫面，點選 [時間與語言] ，在 [語言] 畫面， 點選 [新增語言] ， 在 [選擇要安裝的語言] 畫面，選擇任一種語言，如英文、日文等皆可。
      範例：選擇 English (United States)。
      再按 [下一步] ，[安裝語言功能]畫面 ， 將三個選項 ， 不勾選. 再點選 [安裝]。
      安裝完成後， [語言] 畫面， 即增加了 一種 English(Unitied States) 語言，以上步驟，只需要執行一次即可。
      之後若有出現 IME 已停用的狀況， 即可以再一次切換語言，回預設的語言，此時即可切換為中文輸入法。

  • 當發生報價資訊殘留前一交易日資料時，該處理的動作？

    • 1. 退出程式重新連線。
      2. 若無效請再次結束程式，刪除 SYSTEX 所屬產品資料匣路徑 TICKDATA*.* 後再重新連線即可。

  • 簽屬同意書出現指令碼錯誤時，該如何解決？

    • 2020/6/1 起要簽署 NASDAQ訂戶同意書，若IE版本太舊則會出現「指令碼錯誤」訊息。
      IE8：暫先不彈出同意書簽署視窗，目前尚未支援ie8顯示。
      IE9：可正常顯示同意書簽署視窗。

  • 當安裝後出現「不是正確的win32應用程式」，該如何解決？

    • 清除IE瀏覽器cache，再重新安裝即可。
      由 IE 工具 → 網際網路選項 → 瀏覽歷程記錄: 刪除暫存檔、歷程記錄、Cookie、儲存的密碼及網頁表單資訊，按「刪除」 → 「確定」。

  • 當自設指標刪除時需輸入密碼，是什麼問題造成的？

    • 這問題有可能該指標是老師提供的，並不會提供密碼給客戶：
      1. 請先把原本其他還要保留的指標先匯出。
      2. 找另一台PC重新安裝程式，安裝後，從該台新PC copy Dat auser.study及imtau.ini，覆蓋過去後自設指標就會完全是空的。
      3. 再將之前匯出的指標匯入即可。

      註：若沒有別台PC，可裝在不同資料匣，但桌面捷徑會被覆蓋，記得帶客戶改回，WIN.INI的F_IMIDST_DIRECTORY= 也必需改成原本的路徑即可。

  • 當行情發生快市時，會hang住或畫面會變黑時，該如何改善？

    • 快市會hang住問題，更新於5.0版本之後，富貴2000版本更新在6.08版本。

  • 技術分析出現「歷史資料存放路徑C:SYSTEXNTADATA不存在」，該如何解決？

    • 1. 請查看 WIN.INI>>N_IMIDST_DATA_DIRECTORY 的路徑。
      2. 若存放在 C:SYSTEXNTADATA，請檢查 C:SYSTEXNTADATA 是否存在 (要與 WIN.INI 的路徑一樣)。
      3. 若客戶使用 WIN 7/WIN8，可能是權限問題，請結束程式。用右鍵點選桌面捷徑以系統管理員身份執行;若無效，請將 NTADATA 移動至 D:SYSTEX，並修改 WIN.INI 至相對應的路徑。

  • 啟動程式時會出現「打開系統時指標檔案錯誤」該如何解決？

    • 1. XP 請改「以系統管理員身份登入執行」；WIN VISTA/7/8 請按右鍵點選「以系統管理員身份執行」。
      2. 若無效，請將程式及資料庫安裝至 D:SYSTEX (或其他磁碟)。

  • 設定多組自選股，重新啟動後就不見，該如何解決？

    • 1. 若是 XP 需確認客戶的權限是否為系統管理員，請先由 WINDOWS 左下角 [開始] → [關機] → 查看登出帳號。
      2. 再以 WINDOWS 左下角 [開始] → [控制台] → [使用者帳號] → 帳號是否為「電腦系統管理員」還是「受限制的帳號」，需設為「電腦系統管理員」其儲存檔案的功能才會正常。

  • 連線時出現「Error:PUNTSE.SYM Symbol File計算記憶體大小失敗!」該如何解決？

    • 將電腦的日期及時間修改成正確的日期及時間，或者檢查是否防毒有阻擋，再重新連線即可。

  • 登入程式時帳號密碼無法儲存，需重新輸入，該如何解決？

    • 1. 請以系統管理員身份執行程式，重新輸入帳密連線後連線，再將程式重啟，檢查帳密是否已儲存。
      2. 若無法儲存，結束程式，刪除 C:SYSTEXNIMIDST2003DATUSERPW.DAT，再將程式重啟，檢查 datuserpw 應會重新產生檔案。

      若以上操作無效時，請由防毒軟體設定中將 SYSTEX 資料匣設為排除
      微軟內建 Dender 排除方式：開啟電腦右下角常駐程式中的 Windows Defender → 病毒與威脅防護 → 病毒與威脅防護設定 → 新增或移除排除項目，將 SYSTEX 資料匣設為排除。
      3. 檢查 MirsiteU.ini 內的 SavePassword=1 (若為0,則無法儲存)。

  • 點選程式開啟，但無任何反應改怎麼辦？

    • 1. 請將工作管理員開啟之後，進入「處理程序」將 IManager.exe 結束後，再重新開啟程式。
      2. 若在工作管理員中找不到 Imanager.exe ，請確認電腦時間是否正確，請更正之後再重新安裝程式。

  • 點選技術分析分線時出現「imanager.exe程式發生錯誤」該如何解決？

    • 1.請先嘗試重新安裝程式。
      2.將 ntadata 先更名為 oldntadata ，重建新的目錄 ntadata 後再試一次。若可以再移除重新安裝 (逐筆版本為NTADATAEX)。
      3.若上述無法排除問題，可能是有修改了一些技術分析的參數， 請刪除 C:SYSTEXNIMIDST2003DATIMTAU.INI 重新啟動程式試看看。

  • 開啟程式於載入畫面後，程式就閃退或當掉該如何解決？

    • 1.修改 DAT\manageru.ini 修改 BOOT= 後方清空。
      2.未解決閃退或當掉則，請新安裝程式看看

      若還是無法解決請聯繫客服 02-2363-6322 協助處理。

  • 點選綜、自、期、選...，看不到商品名稱時該如何解決？

    • 請執行系統連結 → 資料下載 → 最新代碼檔下載。
      若無效，可能是電腦日期/時間不對，請修改正確的日期/時間再重要股票檔即可。

      若還是無法解決請聯繫客服 02-2363-6322 協助處理。

  • 檢視公司基本資料時無法顯示或出現「錯誤訊息403」該如何解決？

    • 1. 確認電腦日期時間是否異常。
      2. 重設IE → 桌面左下[開始] → 執行鍵入 inetcpl.cpl 按 Enter ，就會跳出網際網路選項 → 進階 → 重設..win11 沒重設用還原預設。

授權異常

  • 已經是富貴贏家2000專業版用戶，為何看不到部份技術分析創意指標的功能?

    • 富貴贏家2000「經典版」及「專業版」都有'創意指標基本功能'！
      進階創意指標功能 會在 '技術分析' → '選擇指標' → '創意指標' 中會加上 (進階) 的字樣表示有另外加值創意指標！
      且會在 Billing/CRM 有寫 '創意指標' 項目， 若有看到 '創意指標' 的授權，但技術分析看不到， 請刪除 C:SYSTEXFIMIDST2004DatIMAUTHRZ.CTL (此為授權檔) 重連， 若還是無法解決請聯繫客服 02-2363-6322 協助處理。

      註：若是當天開的加值功能授權，隔天才會生效。

  • 當出現到期警示視窗，該如何續訂？

    • 到期前五日開始，每次登入時即會出現警示視窗，若在到期前五日，即有跳出此視窗，請客戶更換到新版。

  • 登入時，彈跳「出未經授權使用！若有問題，請與精業公司聯絡！」訊息，該如何處理？

    • 若客戶未註冊，請客戶至產品網頁[會員] → [開通帳號] 完成會員註冊即可使用。 或輸入錯誤的帳號，清除帳號重新輸入後連線即可。

  • 換版後，啟動時出現「無法找到程序輸入點」，該如何解決？

    • Windows XP 無法支援逐筆新制，更換新版者會出現「無法找到程序輸入點」無法登入情形，請先還原舊版即可恢復連線，並請儘快升級您目前的電腦。

  • 當彈跳出詢問換版，後續步驟為何？

    • 退出程式即可開始進行換版 → 點選設定連結(或系統設定) → 關閉桌面管理員 → 不儲存， 退出程式後再點「更換版本」即可進行更新。

  • 啟動程式後，出現「登錄時失敗，若有問題，請與授權單位聯絡」，該如何解決？

    • 1. 帳號輸入錯誤，清除帳號重新輸入後連線即可。
      2. 客戶帳號到期，故無法登入連線，請連絡業務進行續購。

  • 我該如何重新安裝程式？

    • 官網中點選[支援]，再點 [立即下載]，點下載按鈕即可開始下載安裝。

  • 該如何將工具列加入圖示？

    • 請至工具列中，按下滑鼠右鍵選擇「自訂工具列」，再選擇功能中要加入的圖示按下「加入」至右方使用者設定，按下「確定」即可。
      若要顯示工具列原有的圖示，請在「自訂工具列」開啟後按下左下方的「預設」，再按下「確定」即可。

  • 如何設定多個視窗的畫面？或載入、按快捷鍵叫出多畫面的功能？

    • 1. 先將即時行情、即時新聞、技術分析、基本分析…..等版面設定後，點設定系統連結 → 其他設定 → 多畫面設定，輸入群組名稱，再按「儲存群組」。
      2. 執行系統連結 → 載入畫面配置，選擇要載入的群組名稱即可載入多畫面。
      3. 快速鍵設定：系統連結 → 其他設定 → 多畫面設定點選上方頁籤「快速鍵設定」，選好 F1~F12 其中一個快速鍵，及要對應的畫面群組後，按下方「儲存」，再按確定。要叫出畫面時，一定要先點一下管理員再按下所設定 F1~F12 的快速鍵。

  • 我該如何設定「自選商品」或「股價警示」？

    • 1. 設定自選商品：快速鍵 F12 / 工具列「設」/ 滑鼠右鍵「設定」，點上方「綜合自選商品設定」。
      2. 設定股價警示：進入「綜合自選商品設定」再按「警示」。

  • 技術分析如何儲存常用的指標版面？顯示出已設定的指標版面？

    • 1. F12 設定 → 指標版面配置設定，點選「未定義」(若要覆蓋原有的定義也可)，按「儲存目前配置」再按確定，目前上限可設 50 組。
      2. 按右鍵 → 載入指標版面配置，選擇要載入的版面再按確定。

  • 當磁碟空間不足時，想將目前檔案複製到另一顆硬碟，該如何操作？

    • 如原先在 C:SYSTEXNtadata 改到 D:SYSTEXNtadata。
      執行步驟:
      1. 在D: 先建立 SYSTEX 目錄。
      2. 將 C:SYSTEXNtadata 目錄複製到 D:Systex 下，變成 D:SYSTEXNTADATA。
      3. 修改 win.ini中的 N_IMIDST_DATA_DIRECTORY=C:SYSTEXNTADATA → N_IMIDST_DATA_DIRECTORY==D:SYSTEXNTADATA 。
      4. 測試正常後，可將 C:SystexNtadata 目錄移除。

      註：逐筆版本為 NTADATAE X及 NTADATA 資料夾，請複製二個資料夾。

  • 該如何設定指標參數、畫面顯示、版面格式、儲存版面配置、設定力道條件... 等？

    • 可按右鍵選擇「設定」，也可點工具列中'設'或按快速鍵 F12。
      再選擇上方的頁籤版面格式設定、指標版面配置設定、力道K線條件設定、指標參數設定、均線計算方式設定、畫面顯示設定等功能即可。

  • 該如何備份自選股？

    • 即時行情中「F12」綜合自選商品設定頁籤 → 匯出群組，之後要存入舊資料時再選擇「匯入群組」即可。

  • 報價畫面如何設定排序功能？

    • 請點選綜合 → 選擇「報價資訊(欄位排序)」或「報價/成交警示報價(資訊欄位排序)」 → 點'綜'，直接以滑鼠快點兩下欄位名稱，即可進行排序。

  • 財訊新聞的註記是什麼意思？

    • 新聞註記符號(僅財訊新聞提供):
      1.△(空心三角型向上):偏多+1，代表該則新聞對相關公司/產業屬中性偏正面消息。
      2.▲(實心三角型向上): 利多+2，代表該則新聞對相關公司/產業屬正面消息。
      3.▽(空心三角型向下): 偏空-1，代表該則新聞對相關公司/產業屬中性偏負面消息。
      4.▼(實心三角型向下): 利空-2，代表該則新聞對相關公司/產業屬負面消息。

      註：註記符號只代表該則新聞對個股/產業為正面或負面影響，但與實際股價走勢未必為正相關。

轉檔異常

  • 當即時看盤正常，但資料下載時出現， 「連線失敗，系統找不到指定的檔案」該如何解決？

    • 1.請查看 IE 是否有設離線工作， 若有則改成連線工作即可正常。
      2.請檢查 IE 工具 → 網際網路選項 → 連線 → 區域網路設定，不要勾選任何選項(若公司戶有特殊設定時則為例外)。

  • 當發生缺k線資料時，該如何修正？

    • 1. 執行系統連結 → 資料下載 → 盤後下載，依需要選'自動補齊'或'手動設定。
      2. 若傳完後轉檔轉到一半便停止，請先自行測試CS Lan轉檔。
      3. 若自行測試轉檔正常，則請換連線伺服器再重轉(並通知SE哪一伺服無法轉檔)。
      4. 請將 systexfimidst2004acdwonunzip 目錄之下檔案刪除，再重傳。
      5. 若只缺當月份的資料仍無法轉入，請於有異常的資料匣中將 tmplog.dat及IndexFile 資料匣下方的檔案刪除。
      (Ex:國際指數異常，請將 idxdly/tmplog.dat 及 idxdly/IndexFile/Tmplog.idx 的檔案刪除) 後再重轉即可。 6. 若逐筆版本有缺資料， 請先確認缺的是哪一種資料， 再到對應的資料夾刪【NTALog.資料夾名稱】與【資料夾名稱.db】，再將缺的資料重轉即可。 例如: 缺日檔，請到C:SystexNTADATAEX。

  • 盤後下載出現「無法建立與伺服器連線」該如何解決？

    • 可能有以下原因:
      1. 伺服器 web server 異常，請客服人員測試連同一伺服器是否有相同情況。
      2. IE proxy 設定(通常都不用打勾，除非客戶端使用的是公司(特定)網路，需要特別設定再依該公司的Proxy 設定為準)。
      3. 個人防火牆攔截，請設排除後再重轉(可參閱M:1公用事務區產品服務防毒軟體&防火牆設定)。
      4. 若都檢查正常，可重裝程式再重轉。

  • 盤後下載時發生 imacd.exe 發生錯誤，該如何解決？

    • 1. 客戶執行資料下載時，出現「IMACD.EXE 應用程式錯誤」.請確認客戶的電腦時間，有沒有設錯。
      2. 請重新安裝程式再試測試。
      3. 請更名 NTADATA 資料夾，建立新資料夾 -NTADATA，再重新資料下載看看 (逐筆版本則須更名NTADATAEX及NTADATA)。

  • 無法查看操盤密碼網頁，該如何解決？

    • 1.詢問客戶使用情況，例平常看那個頁面，使用頻率..等。 2.告知:目前工程師處理中。

  • 之前基本分析有財訊每季評估，現在怎麼消失了？

    • 原本【財訊股市總覽】，已更新出版【四季數報】。
